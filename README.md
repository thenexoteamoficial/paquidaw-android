# 🎮 Paquidaw Android

**Versión 100% funcional para Android** del videojuego Paquidaw

Desarrollado con **Python + Kivy** para que puedas crear un APK real fácilmente.

## ✨ Características de esta versión Android

- Juego 2D jugable con controles táctiles
- Sistema de puntuación
- Enemigos que aparecen
- Efectos de sonido básicos (puedes mejorar)
- Totalmente funcional en Android

## 🚀 Cómo crear el APK (paso a paso)

### Requisitos en tu computadora:

```bash
pip install buildozer
pip install cython
```

### Pasos:

1. Clona este repositorio
2. Entra a la carpeta
3. Ejecuta:
   ```bash
   buildozer init
   buildozer android debug
   ```
4. El APK aparecerá en `bin/`

> **Nota**: La primera compilación puede tardar 10-20 minutos.

## 💻 Ejecutar en PC (para probar)

```bash
pip install kivy
python main.py
```
